Vibe Check API

A REST-ish API for determining the vibe of literally anything. Send us a query, we send back a score, a vibe, a confidence level, and a recommendation you definitely should not follow.

No auth, no rate limits, no backend, no consequences.

Authentication

There is none. We tried adding API keys but honestly the vibes are public information. If your security team asks, tell them it's open by design.

Rate limits

Technically unlimited, practically limited by how fast you can move your mouse. The server (your browser) can handle it.

GET /vibe-check

Returns a full vibe assessment for the given query. The assessment is generated using a proprietary blend of Math.random() and confidence.

GET /vibe-check?q={your_query}
ParamTypeRequiredDescription
q string no, but encouraged The thing whose vibe you'd like checked. A name, a decision, a life choice, whatever.
Example request 
curl "https://vibe.xevrion.dev/vibe-check?q=quitting+my+job+to+make+an+api+that+does+nothing"
Example response - 200 OK 
{
  "query": "quitting my job to make an api that does nothing",
  "vibe": "immaculate",
  "score": 94,
  "confidence": "suspiciously high",
  "recommendation": "keep doing whatever you're doing bhai",
  "red_flags": 0,
  "certified": true,
  "checked_by": "the vibe council",
  "timestamp": "2026-06-12T03:14:07.029Z"
}

Response schema

FieldTypeDescription
querystringEcho of whatever you sent in q. We don't store it. We don't even look at it.
vibestringOne of 20+ official vibe classifications, chosen at random.
scoreinteger0–100. Not based on anything. Do not cite this in a report.
confidencestringHow sure the council is. Spoiler: never very.
recommendationstringWhat you should do next. Following this is at your own risk.
red_flagsinteger0–5. Counted by vibe, not by evidence.
certifiedbooleanWhether this vibe check is "certified". Certification body: see checked_by.
checked_bystringThe authority responsible for this result. Often not a real authority.
timestampstringISO 8601. The one real thing in this entire response.

Errors

There are no error states. Every query, including empty ones, gibberish, and unicode emoji soup, returns 200 OK with a full vibe report. The vibe council does not believe in 400s.
